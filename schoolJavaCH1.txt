4)쓰레드 개념
순차프로그램과 유사하게 시작 실행 종료의 순서를 가진 제어의 흐름임
멀티스레딩 시스템이란 이와 같은 스레드가 하나의 프로그램 내에 여러개 존재 할 수 있는 시스템이다.
자바에서 스레드는 하나의 객체이다.
new Thread(); 스레드 객체 생성후 실제로 start() 메소드 호출후 스레드 실행 개시한다. 이때 자바 시스템에는 run() 메소드를 호출하여 스레드를 실행 상태로 만든다.
그결과 start, run 메소드가 병행으로 실행된다. 그리고 run 메소드 실행 완료 후 새로운 스레드 종료
run 메소드는 기존 자바에서 실행 코드가 없으니 재정의하여 프로그래머가 쓰도록 코딩해야한다.
